17/**
18 * Public facade for parsing MongoDB Query DSL expressions into Rules.
19 *
20 * Provides a clean interface for converting MongoDB-style query documents
21 * into executable Rule objects. This class follows the standardized DSL facade
22 * pattern that should be replicated across all DSL implementations.
23 *
24 * Pattern: Each DSL should provide three public classes:
25 * - {DSL}Parser: Parse DSL strings → Rule objects
26 * - {DSL}Serializer: Serialize Rule objects → DSL strings
27 * - {DSL}Validator: Validate DSL strings without full parsing
28 *
29 * Example usage:
30 * ```php
31 * $parser = new MongoQueryParser();
32 * $rule = $parser->parse(['age' => ['$gte' => 18]]);
33 * $result = $rule->evaluate($context);
34 *
35 * // Or from JSON string:
36 * $rule = $parser->parseJson('{"age": {"$gte": 18}}');
37 * ```
38 *
39 * @author Brian Faust <brian@cline.sh>
40 *
41 * @see MongoQueryRuleBuilder Internal implementation used by this facade
42 * @see MongoQuerySerializer For converting Rules back to MongoDB query documents
